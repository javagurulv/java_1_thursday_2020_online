

    Ответы:

    1. Для чего применяются объекты этого класса?
    Optional - это класс оболочка, которая внутри себя содержит некоторое значение,
    которое может быть NULL, и, если это значение является NULL, может предпринять
    какие-то действия.

    2. Какие два состояния могут быть у объектов этого класса?
    NULL, NON_NULL.

    3. Как создаются экземпляры данного класса?
    Класс Optional нельзя создать прямым образом через new. Он имеет приватный конструктор.
    Но для его создания существуют статические методы:

        Optional.empty();
        Optional.of(T value);
        Optional.ofNullable(T value);

    4. Как можно проверить в каком состоянии находится экземпляр данного класса?
    С помощью метода isPresent() - true, если значение есть, false, если значение = NULL;

    5. Как можно получить объект, который находится внутри Optional?
    С помощью метода get().
