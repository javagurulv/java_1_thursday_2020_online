1. С помощью File Explorer зайти в папку,
в которой лежит файл .java, который нужно
скомпилировать.
2. Вводим в адресной строке системную команду
“cmd” и нажимаем клавишу Enter на клавиатуре.
3. Для того, чтобы скомпилировать файл программы
нужно набрать в консоли команду javac и в качестве
параметра передать имя нашего файла.
Эта команда вызовет компилятор, который создаст
файл Main.class, содержащий скомпилированный
код нашей java программы.
