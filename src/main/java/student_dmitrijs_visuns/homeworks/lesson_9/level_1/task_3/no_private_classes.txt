Возможно ли в Java объявить класс и сделать его приватным,
не видимым для всего другого кода? Подумайте почему?
Ответ на этот вопрос закомитьте в текстовом файле.

Класс нельзя сделать приватным, так как это будет означать, что он не виден для любых других классов, не имеет функционала.

Среда разработки не позволит использовать модификатор доступа private по отношению к классу.
