Найти в интеренете ответ на вопрос:
Как откомпилировать код с помощью JDK компилятора javac,
без помощи среды разработки (из консоли)?

Ответ:

Сначала надо в консоли перейти в папку с исходным кодом
(для того, чтобы не набирать в командной строке весь длинный путь к файлу).
Затем запустить компилятор javac NameOfTheCode.java.
Компилятор создает готовый к исполнению на JVM код NameOfTheCode.class,
который можно запустить на выполнение: java NameOfTheCode

Пример:

C:\Java\JABGN>javac HelloWorld.java

C:\Java\JABGN>java HelloWorld
Hello World!
