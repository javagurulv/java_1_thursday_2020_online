Упростите метод boolean isEven(int number).
Перед изменением кода убедитесь, что у вас есть автоматические
тесты для проверки этого кода. Если автоматических тестов нет
то напишите их, убедитесь, что они проходят и только тогда меняйте код.

class Calculator {

    public boolean isEven(int number) {
         if(number % 2 == 0) {
             return true;
         } else {
             return false;
         }
    }

}

Подсказка: обратите внимание на тип возвращаемого параметра
и на условие, которое записано в if операторе.
